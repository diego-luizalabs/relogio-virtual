Projeto para Mentoria

Este projeto foi desenvolvido com o objetivo de criar uma página com informações sobre mim e o que gosto de fazer.

O projeto foi desenvolvido utilizando HTML e CSS, com o objetivo de praticar as habilidades adquiridas durante o curso de Desenvolvimento de Sistemas.

Sobre mim

Meu nome é Diego Santos, tenho 17 anos. Atualmente trabalho no Luizalabs na área de Suporte Técnico. Comecei a programar com 8 anos, fazendo meu primeiro curso de Desenvolvimento de Jogos usando o Unity. Após concluir o curso, fiz uma pausa, pois percebi que não era algo que me interessava tanto. Depois de 9 anos, retomei meu interesse pela programação e estou aprendendo novas tecnologias para aprimorar minhas habilidades e, quem sabe, seguir minha carreira nessa área.

O que gosto de fazer

Gosto de aprender novas tecnologias e linguagens de programação. Atualmente estou aprendendo HTML, CSS e JavaScript. Além disso, gosto de jogar videogame, assistir séries e filmes, e praticar esportes. Estou sempre em busca de novos desafios e oportunidades para crescer profissionalmente. Acredito que essa mentoria será uma ótima oportunidade para aprender e compartilhar conhecimentos com outros profissionais da área.
